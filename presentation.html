Gestione Punti di Ripristino

Tool grafico per Windows 11

PowerShell 5.1+ WinForms GUI Open Source

Giugno 2026

Il Problema

Windows non offre un'interfaccia semplice per gestire i singoli punti di ripristino.

❌ Interfaccia nativa limitata

Protezione Sistema permette solo di eliminare tutti i punti o crearne di nuovi. Nessuna gestione selettiva.

❌ vssadmin complesso

Da terminale si può, ma richiede comandi manuali e conoscenza degli UUID dei shadow copy.

❌ Nessuna personalizzazione

Impossibile aggiungere descrizioni significative ai punti per identificarli facilmente.

❌ Rischio errori

Un comando sbagliato via terminale può cancellare dati irreversibilmente.

Obiettivi del Progetto

  • GUI nativa senza dipendenze esterne
  • Eliminazione selettiva dei punti di ripristino
  • Creazione guidata con un click
  • Descrizioni personalizzate per ogni punto
  • Sicurezza: niente command injection, niente vulnerabilità LPE
  • Funziona su Windows 10 e 11, nessuna installazione richiesta

La Soluzione

📋 Elenco punti

Carica tutti i restore point da vssadmin, mostra data, descrizione e UUID in una ListView chiara.

🗑️ Eliminazione selettiva

Seleziona un punto e cancellalo con conferma. Validazione UUID rigida prima di ogni operazione.

➕ Creazione

Crea un nuovo punto con Checkpoint-Computer, operazione tracciata negli audit log.

✏️ Descrizioni custom

Modifica la descrizione di qualsiasi punto. Salvata in HKCU\Software (Registry protetto), non in file temporanei.

Architettura

  • Frontend: Windows Forms (PowerShell 5.1)
  • Backend: vssadmin.exe via System.Diagnostics.Process
  • Storage descrizioni: Registry → HKCU\Software\OttimizzaWindows\ShadowCopyDescriptions
  • Audit: Windows Event Log → sorgente OttimizzaWindows
  • Admin check: WindowsPrincipal.IsInRole, auto-elevazione con RunAs

Sicurezza

Analisi statica completata — tutte le vulnerabilità corrette.

✅ Command Injection

Validazione UUID con regex prima di ogni chiamata. Blocco con audit log se rilevato tentativo.

✅ niente cmd.exe

Process.Start diretto senza shell intermedi. Nessun parsing di caratteri speciali.

✅ niente temp file

Descrizioni salvate nel Registry (HKCU). Nessun file in %TEMP% vulnerabile a LPE.

✅ Timeout & Audit

Processi vssadmin con timeout 30s. Ogni operazione tracciata su Windows Event Log.

Utilizzo

Esecuzione

.\GestionePuntiRipristino.ps1

Oppure da Explorer: tasto destro → Esegui con PowerShell (come Amministratore).

Requisiti

  • Windows 11 / Windows 10
  • PowerShell 5.1+
  • Amministratore locale
