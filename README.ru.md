# GestionePuntiRipristino

Графический инструмент для Windows 11 для создания, просмотра, изменения и удаления точек восстановления системы.

## Требования

- **Windows 11** (также работает на Windows 10)
- **PowerShell 5.1+**
- **Запуск от имени Администратора** (требуется для взаимодействия с Volume Shadow Copy)

## Установка

Скачайте `GestionePuntiRipristino.ps1` и запустите от имени Администратора:

```powershell
.\GestionePuntiRipristino.ps1
```

Или в Проводнике: правый клик → **Запуск с PowerShell**.

## Возможности

- **Список точек** — показывает все существующие точки с датой и описанием
- **Удаление** — удаляет конкретную точку с подтверждением
- **Создание** — создаёт новую точку восстановления
- **Редактирование описания** — настройка описаний (сохраняются в HKCU\Software\OttimizzaWindows)
- **Запуск восстановления** — открывает нативный интерфейс Windows (`rstrui.exe`)

## Безопасность

- Строгая проверка UUID перед каждой операцией с `vssadmin`
- Процессы запускаются через `Process.Start` без промежуточной оболочки
- Тайм-аут 30 секунд предотвращает зависания
- Аудит в Windows Event Log (источник: `OttimizzaWindows`)
- Описания хранятся в реестре (HKCU), а не во временных файлах
